Simple Web Calculator
Description
A basic web-based calculator built with vanilla HTML, CSS, and JavaScript, designed to perform simple arithmetic operations. This project provides a clean and intuitive interface for users to calculate results.

Technologies Used
HTML: For structuring the content of the calculator
CSS: For styling and layout of the calculator interface
JavaScript: For adding interactive functionality to the calculator

Features
Basic Arithmetic Operations: Addition, subtraction, multiplication, and division
Clear Button: Resets the calculator for new calculations
Equals Button: Calculates the result of the current operation
Simple Interface: Easy to use and understand for users of all levels

Setup Instructions
To view and use the calculator, you can:

Open the index.html file in a web browser
Visit the live deployment on Vercel: https://simple-calculator-ten-gilt.vercel.app/
